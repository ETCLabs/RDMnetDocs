RdmUid uid

The client's UID.

If the client has a static UID, fill in the values normally. If a dynamic UID is desired, assign using RPT_CLIENT_DYNAMIC_UID(manu_id), passing your ESTA manufacturer ID. All RDMnet components are required to have a valid ESTA manufacturer ID.
